# Valentine mini-site (QR)

## Что это
Одностраничный романтический сайт с вашей музыкой.
Открывается по ссылке (её потом зашиваешь в QR).

## Файлы
- index.html
- styles.css
- script.js
- music.m4a (+ music.mp3)
- (можешь добавить фото: photo1.jpg, photo2.jpg, ...)

## Как добавить ваши фото
Самый простой способ: заменить блоки "Фото 1/2/3" на `<img>`.

В `index.html` найди:

<div class="ph ph1">Фото 1</div>

и замени на, например:

<img class="ph ph1" src="photo1.jpg" alt="Фото 1">

Потом просто положи `photo1.jpg` рядом с index.html.

## Как выложить в интернет (2 варианта)
### Вариант A — GitHub Pages
1) Создай репозиторий (Public) и загрузи туда все файлы из папки.
2) Settings → Pages → Deploy from branch → main / root.
3) Получишь ссылку вида: https://USERNAME.github.io/REPO/

### Вариант B — Netlify (проще)
1) Зайди в Netlify → Add new site → Deploy manually.
2) Перетащи папку/zip.
3) Netlify даст ссылку сразу.

## QR-код

## Важно про автоплей
На телефонах музыка почти всегда запускается только после нажатия (это норма браузеров).
